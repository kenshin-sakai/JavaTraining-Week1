## 工夫した点・学んだこと
工夫というより今回心掛けたことは、生成AIの力を極力使わないこと、ネットで調べるよりも教科書を読みこんでいろいろ試すことである。
環境構築の部分は検索並びに生成ＡＩを頼ってしまったが、プログラムの作成に関してはそれを一切せずに完成させた。
工夫した点は、結果の表示の際配列を使って文字数を削減した点、String judgeの際負けの手をelseにすることで処理を軽くした点である。
学びとしては、returnの使い方である。最初はリターンをメゾットにつき一回しか使えないと勘違いしており、コードが少し長くなってしまっていたが、
教科書を読みなおし、一回しか使えないわけではなく、一回しか発動しないということを気づき、その後修正を施した。
